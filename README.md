# DevRel Tutorial Maker

An AI-powered tool for generating tutorials and guides, designed to help Developer Advocates create high-quality documentation efficiently.

## Installation

1. Ensure you have Python 3.9 or higher installed
2. Clone this repository
3. Create and activate a virtual environment:

   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Unix/macOS
   # or
   .venv\Scripts\activate  # On Windows
   ```

4. Install the package:

   ```bash
   pip install -e .
   ```

## Usage

After installation, you can use the `drdemo` command to start generating tutorials.
